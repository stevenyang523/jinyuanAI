# AI 投研助手

一个面向研究报告编制、数据同步、内容校对、历史报告维护及内部提交流程的AI投研助手。

## 功能介绍

### Word 报告数据同步

- 支持从 Excel 工作簿读取数据并写入 Word 模板。
- 支持同步报告日期、评级、分析师、研究助理及其他结构化字段。
- 支持公司报告和行业报告两种模式。
- 支持通过 Word 内容控件更新首页字段、财务表格和报告元数据。

### 模板与估值模型管理

- 可直接从启动器打开公司报告和行业报告模板。
- 支持按市场打开公司估值模型：
  - A 股
  - 港股
  - 美股
- 支持打开行业估值模型。
- 可根据文件名自动查找匹配的 Word 模板和 Excel 模型文件。

### 人员库与历史报告管理

- 支持从本地人员库加载分析师和研究助理信息。
- 支持维护历史报告记录。
- 可根据分析师、公司、行业和报告类型匹配并更新相关报告。
- 在覆盖本地人员库前自动生成备份文件。

### AI 智能助手

- 支持配置 OpenAI 兼容或 Anthropic 兼容的 AI API。
- 支持 AI 智能纠错。
- 支持基于当前文档内容进行智能问答。
- API 配置信息保存在用户本机。

### 提交报告测试功能

- 读取 Word 文档中报告标题和报告首页内容。
- 在内嵌浏览器中打开内部 OA 页面。
- 支持用户先完成登录，再手动触发自动填写。
- 自动填写网页中的标题和主要内容字段。
- 尽可能将 OA 新打开的页面保留在内嵌浏览器中。
- 支持网页前进和返回。
- 支持折叠式浏览器工具栏，减少对网页内容的遮挡。

### OA 登录辅助

- 用户确认后，可将 OA 用户名和密码保存在本机，登录信息加密保存。
- 再次打开 OA 登录页时，可自动填写已保存的账号和密码。
- 支持随时清除本地保存的登录信息。
- 不会自动识别或绕过验证码。
- 不会自动点击最终提交或审批按钮。

### VPN 与网络检测

- 支持检测 EasyConnect 相关进程是否正在运行。
- 未检测到 VPN 客户端时，可提示用户登录 VPN 或确认已连接公司内网。
- 支持自动启动 EasyConnect。
- 网络状态确认后再打开提交报告窗口。
